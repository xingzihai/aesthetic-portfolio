# 溶 · 萤火 (Melt × Firefly)

「溶」与「浮光萤火」的混合方案

---

## 设计理念

**核心意象**：冰水中溶解的"氷"字符号，周围漂浮着萤火虫般的光点

**融合策略**：
- 结构：「溶」的4圆结构（"氷"字骨架）
- 粒子：「萤火」的随机漂浮光点
- 交互：融化效果 + 躲避效果
- 色彩：冰蓝灰核心 + 暖粉融化 + 黄绿萤火

---

## 结构

```
        ○ ← 上圆（冰晶）
       ╱ ╲
      ○   ○ ← 左右圆（水滴）
       ╲ ╱
        ○ ← 下圆（融化核心）
        
  ○   ○   ○   ○   ○   ← 游离萤火粒子
    ○       ○       ○
```

---

## 交互效果

| 交互 | 效果 |
|------|------|
| 鼠标接近 | 下圆融化放大+模糊 |
| 鼠标移动 | 萤火粒子躲避 |
| 点击 | 涟漪扩散（双色） |
